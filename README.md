# PharmaLink - Hackathon Submission

## Overview
PharmaLink is a web application that enables digital prescription transfer between doctors and pharmacies. This is a demo-only version â€” no real patient data is used.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask, Flask-CORS
- Database: SQLite (demo)
- Version Control: Git/GitHub
- Deployment: Render.com (optional)

## Features
- User login & registration
- Doctors can create medication transfer requests
- Pharmacists can approve and complete requests
- Dashboard displays pending and completed requests
- Secure (demo only, passwords hashed, JWT authentication)
